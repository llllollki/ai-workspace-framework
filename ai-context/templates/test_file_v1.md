# Test File Template — v1

## Responsibility

Use this template when documenting test coverage expectations for a feature or module.

## Template

```markdown
# Tests — [Feature or Module Name]

## Scope

[What is covered by these tests and what is intentionally excluded.]

## Test Cases

### [Test group 1]

- [ ] [test case description]
- [ ] [test case description]

### [Test group 2]

- [ ] [test case description]

## Edge Cases

- [ ] [edge case to cover]

## Out of Scope

[Scenarios intentionally not tested here and why.]

## Notes

[Any setup requirements, fixtures, or non-obvious test behavior.]
```
