# Design System

## Responsibility

This file documents shared visual design tokens, typography, color palette, spacing, and layout conventions that apply across projects in this workspace.

## What Belongs Here

- Color tokens and palette
- Typography scale and font choices
- Spacing and sizing scales
- Layout grid conventions
- Motion and animation guidelines
- Icon set reference

## What Does Not Belong Here

- Component-level implementation details (see `ui_components.md`)
- Project-specific overrides or one-off styles
- CSS framework configuration (belongs in the project repo)

## Related Files

- `ui_components.md` — component patterns that use this design system
- `coding_standards.md` — code-level conventions
